# Bullets

The Bullets component displays a set of bullets, which allow users to alternate between different screens.

## Import

```tsx
import { Bullets } from '@faststore/ui'
```

## Usage

```tsx live
<BulletsTemplate
  activeBullet={3}
  onClick={(e, idx) => {}}
  totalQuantity={5}
/>
```
