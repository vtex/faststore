# LoadingButton

Loading Buttons are used to disable interactions or provide a visual cue that an action is being processed.

## Import

```tsx
import { LoadingButton } from '@faststore/ui'
```

## Usage

```tsx
<LoadingButton loading='true'>
    Loading Button
</LoadingButton>
```

## Props

<PropsComponent
    name="loading"
    type= "boolean"
    description="The current loading state of the button."
    required="true"
/>
<PropsComponent
    name="testId"
    type= "string"
    description="ID to find this component in testing tools (e.g.: cypress, testing library, and jest)."
    defaultValue="store-loading-button"
/>

## Customization

`data-store-loading-button`

This component inherits [Button](/reference/ui/atoms/Button) css selectors.
