# SearchInput

The Search Input component allows users to enter an input and query results from a database.

## Import

```tsx
import { SearchInput } from '@faststore/ui'
```

## Usage

```tsx live
<SearchInput
  placeholder="Search"
/>
```

## With custom icon

```tsx live
<SearchInput
  icon={<CustomIcon />}
  placeholder="Search"
/>
```
