# Spinner

Spinners provide a visual cue that an action is being processed.

## Import

```tsx
import { Spinner } from '@faststore/ui'
```

## Usage

```tsx live
<Spinner />
```

## Props

<PropsComponent
    name="testId"
    type= "string"
    description="ID to find this component in testing tools (e.g.: cypress, testing library, and jest)."
    defaultValue="store-spinner"
/>

## Customization

`data-store-spinner`
