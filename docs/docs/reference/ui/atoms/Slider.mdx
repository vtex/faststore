# Slider

Sliders allow users to select a single value from a range.

## Import

```tsx
import { Slider } from '@faststore/ui'
```

## Usage

```tsx live
<Slider
  max={500}
  min={0}
/>
```

## Props

<PropsComponent
    name="testId"
    type= "string"
    description="ID to find this component in testing tools (e.g.: cypress, testing library, and jest)."
    defaultValue="store-slider"
/>

## Customization

`data-store-slider`

`data-store-slider-range`

`data-store-slider-thumb='(left|right)'`
