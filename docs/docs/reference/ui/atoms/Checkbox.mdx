# Checkbox

Checkboxes allow users to toggle an option on or off.

## Import

```tsx
import { Checkbox } from '@faststore/ui'
```

## Usage

```tsx live
<Checkbox
  checked
  onClick={function noRefCheck(){}}
/>
```
