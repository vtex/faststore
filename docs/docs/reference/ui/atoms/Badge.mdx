# Badge

Badges are status descriptors used to highlight a relevant content.

## Import

```tsx
import { Badge } from '@faststore/ui'
```

## Usage

```tsx live
<Badge>-25%</Badge>
```

## Props

<PropsComponent
    name="testId"
    type= "string"
    description="ID to find this component in testing tools (e.g.: cypress, testing library, and jest)."
    defaultValue="store-badge"
/>

## Customization

`data-store-badge`
