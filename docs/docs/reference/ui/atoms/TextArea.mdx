# TextArea

TextArea components allow users to enter and edit text in a given field.

## Import

```tsx
import { TextArea } from '@faststore/ui'
```

## Usage

```tsx live
<TextArea
  defaultValue="Write something here"
/>
```

## Variant

```tsx live
<>
<TextArea
  defaultValue="Success"
  variant="success"
/>
<br/>
<TextArea
  defaultValue="Error"
  variant="error"
/>
</>
```
