# useCart

The `useCart` hook gives you access to the cart, as well as helper functions for applying changes to it, like `addToItem`, `removeItem` etc.

## Import

```tsx
import { useCart } from '@faststore/sdk'
```

## Usage

```tsx
function Cart () {
  const { items, isValidating } = useCart()

  return (
    <>
      {items.map(item => (
        <div>name: {item.itemOffered.name}</div>
        <div>Price: {item.price}</div>
      ))}
      <a href="/checkout">{isValidating ? 'loading...' : 'Checkout'}</a>
    </>
  )
}
```
