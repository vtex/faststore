# Tiles (Matter UI)

## Intention
Responsive tiles grid layout.

## Description
Tiles component for creating grid layouts with responsive columns.

## Import
```tsx
import { Tiles } from 'src/components/ui/Tiles'
```

## Examples

```tsx
<Tiles>
  <Tile>
    <img src="/category1.jpg" alt="Electronics" />
    <h3>Electronics</h3>
  </Tile>
  <Tile>
    <img src="/category2.jpg" alt="Clothing" />
    <h3>Clothing</h3>
  </Tile>
</Tiles>
```
