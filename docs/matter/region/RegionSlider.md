# RegionSlider (Matter)

## Intention
Mobile slide-out panel for region selection.

## Description
RegionSlider provides mobile-optimized region selection interface.

## Import
```tsx
import RegionSlider from 'src/components/region/RegionSlider'
```

## Examples

```tsx
<RegionSlider
  isOpen={isOpen}
  onClose={() => setIsOpen(false)}
/>
```
