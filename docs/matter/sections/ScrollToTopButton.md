# ScrollToTopButton Section (Matter)

## Intention
Floating button to scroll to page top.

## Description
ScrollToTopButton appears after scrolling down, scrolls page to top on click.

## Import
```tsx
import ScrollToTopButton from 'src/components/sections/ScrollToTopButton'
```

## Examples

```tsx
<ScrollToTopButton />
// Automatically shows after scrolling 400px
```
