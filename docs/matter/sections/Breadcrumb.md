# Breadcrumb Section (Matter)

## Intention
Page context breadcrumb navigation section.

## Description
Breadcrumb section with automatic page hierarchy detection and Next.js routing.

## Import
```tsx
import Breadcrumb from 'src/components/sections/Breadcrumb'
```

## Data Integration

- Page context from Next.js router
- Automatic breadcrumb generation

<Breadcrumb />
// Auto-generates: Home / Category / Product
