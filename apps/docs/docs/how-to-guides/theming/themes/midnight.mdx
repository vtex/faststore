# Midnight

A darkmode theme created for testing purposes.

![Midnight theme](https://vtexhelp.vtexassets.com/assets/docs/src/midnight___23f91a5508d29e7e8a7ec7a209b81fe6.webp)

## Usage

1. Import the `midnight.scss` file to the `src/pages/_app.tsx` file.
```jsx
import '../styles/themes/midnight.scss'
```
2. Save your changes and go to the `store.config.js` file.
3. Replace the value of `theme` with `midnight`:
   ```js
   // Theming
   theme: 'midnight',
   ```
4. Save your changes and go to `src/fonts/WebFonts.tsx`.
5. Add the [Raleway](https://fonts.google.com/specimen/Raleway?query=raleway) font to `WebFonts` as in the following:
   ```tsx
   function WebFonts() {
     return (
       <link
         rel="stylesheet"
         href="https://fonts.googleapis.com/css2?family=Raleway:wght@300;400;700;900&display=swap"
       />
     )
   }
   ```
6. Save your changes.
