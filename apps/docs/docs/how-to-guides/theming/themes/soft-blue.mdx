# Soft Blue

A basic theme created for testing purposes.

![Soft blue theme](https://vtexhelp.vtexassets.com/assets/docs/src/soft-blue___17c37d94c9840b6d761cb4247be7fd9f.webp)

## Usage

1. Import `soft-blue.scss` file on `src/pages/_app.tsx`

```jsx
import '../styles/themes/soft-blue.scss'
```

2. Go to `store.config.js` and change the `theme` to your file's name:

```js
// Theming
theme: 'soft-blue',
```

3. Add the [Lato](https://fonts.google.com/specimen/Lato?query=lato) font on `src/fonts/WebFonts.tsx`

```tsx
function WebFonts() {
  return (
    <link
      rel="stylesheet"
      href="https://fonts.googleapis.com/css2?family=Lato:wght@300;400;700;900&display=swap"
    />
  )
}
```
