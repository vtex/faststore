# Soft Blue

A basic theme created for testing purposes.

![Soft blue theme](https://vtexhelp.vtexassets.com/assets/docs/src/soft-blue___17c37d94c9840b6d761cb4247be7fd9f.webp)

## Usage

1. Import the `soft-blue.scss` file to the `src/pages/_app.tsx` file.
   ```jsx
   import '../styles/themes/soft-blue.scss'
   ```
2. Save your changes and go to the `store.config.js` file.
3. Replace the value of `theme` with `soft-blue`:
   ```js
   // Theming
   theme: 'soft-blue',
   ```
4. Save your changes and go to `src/fonts/WebFonts.tsx`.
5. Add the [Lato](https://fonts.google.com/specimen/Lato?query=lato) font to `WebFonts` as in the following:
   ```tsx
   function WebFonts() {
     return (
       <link
         rel="stylesheet"
         href="https://fonts.googleapis.com/css2?family=Lato:wght@300;400;700;900&display=swap"
       />
     )
   }
   ```
6. Save your changes.
