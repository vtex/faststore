# CheckboxField

CheckboxField wraps a [Checkbox](/reference/ui/atoms/Checkbox) input and its corresponding [Label](/reference/ui/atoms/Label). It allows users to select one option from a set.

## Overview

```tsx live
<div className="overviewSection">
  <CheckboxField label="Default" id="checkboxfield-default" />
  <CheckboxField label="Checked" id="checkboxfield-checked" checked readOnly />
  <CheckboxField label="Disabled" id="checkboxfield-disabled" disabled />
  <CheckboxField
    label="Checked & Partial"
    id="checkboxfield-checked-partial"
    checked
    partial
    readOnly
  />
</div>
```

---

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { CheckboxField } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/molecules/CheckboxField/styles.scss'
```

## Usage

<CheckboxField label="Faststore" id="faststore" />
