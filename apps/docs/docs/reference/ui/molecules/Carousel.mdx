# Carousel

Carousels are a set of rotating banners. They contain navigational controls which allow users to alternate between multiple images and content in the same section.

## Import

```tsx
import { Carousel } from '@faststore/ui'
```

## Usage

```tsx live
<Carousel
  controls="complete"
  transition={{
    duration: 400,
    property: 'transform'
  }}
>
  <img
    alt=""
    height={614}
    src="https://storecomponents.vtex.app/assets/fit-in/1280x613/center/middle/https%3A%2F%2Fstorecomponents.vtexassets.com%2Fassets%2Fvtex.file-manager-graphql%2Fimages%2F331632a0-fa52-4f08-8e45-df762d97a289___167e4c8385c3129b1a2ddab9156510ba.jpg"
    width="100%"
  />
  <img
```
