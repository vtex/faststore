# LinkButton

`LinkButton` is similar to a button, however acts as a link to navigate users between pages.

## Overview

```tsx live
<LinkButton href="#">Link Button</LinkButton>
```

---

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { LinkButton } from '@faststore/ui'
```

---

## Usage

```tsx live
<LinkButton href="#">Link Button</LinkButton>
```

## Customization

For further customization, you can use the following data attributes:

`data-fs-link-button`
