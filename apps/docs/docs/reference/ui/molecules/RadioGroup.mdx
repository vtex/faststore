# RadioGroup

The Radio Group component allows users to select a single option from a list of two or more mutually exclusive options.

## Import

```tsx
import { RadioGroup, RadioOption } from '@faststore/ui'
```

## Usage

```tsx live
function Component () {
  const [option, setOption] = useState('radio-1')

  return (
    <RadioGroup
      name="radio-group"
      selectedValue={option}
      onChange={(v) => setOption(v.currentTarget.value)}
    >
      <RadioOption value="radio-1" label="Radio 1">
          <span>Radio 1</span>
      </RadioOption>
      <RadioOption value="radio-2" label="Radio 2">
          <span>Radio 2</span>
      </RadioOption>
    </RadioGroup>
  )
}
```
