# BuyButton

BuyButton intends to send the user to the checkout/cart step directly.

## Overview

```tsx live
<BuyButton>Buy Now</BuyButton>
```

---

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { BuyButton } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/atoms/BuyButton/styles.scss'
```

---

## Usage

```tsx live
<BuyButton>Buy Now</BuyButton>
```
