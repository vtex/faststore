ProductCard displays summarized information about a product.

The `ProductCard` is a compound of the following:
- `ProductCardImage`: wraps the product's image.
- `ProductCardContent`: wraps the content's details.
- `ProductCardActions`: wraps the call-to-action button of the product card.

## Import

```tsx
import {
  ProductCard,
  ProductCardActions,
  ProductCardContent,
  ProductCardImage,
} from '@faststore/ui'
```

### Use cases
Use the `ProductCard` in the following context:
- Product Collections on the home page.
- Product availability statuses on the home pages, e.g., *Just Arrived*.
- Carousel with product promotions.

## Best practices

### Do's
- Use high-quality product images.
- Write the product's title in two lines.
- For the call-to-action button, use action verbs such as *Buy*, *Save*, and *Add to cart*.

### Don'ts
- Do not use the `ProductCard` for empty state scenarios. If the case, use the `Skeleton` component.
- Do not use more than three badges (if necessary).

## Accessibility
- The `img` elements must have an `alt` attribute. 
- For images containing text, the `alt` description must includes the image's text.
