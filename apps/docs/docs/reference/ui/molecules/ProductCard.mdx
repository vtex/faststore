ProductCard displays summarized information about a product.

The `ProductCard` is a compound of the following:
- `ProductCardImage`: wraps the product's image.
- `ProductCardContent`: wraps the content's details.
- `ProductCardActions`: wraps the call-to-action button of the product card.

## Import

```tsx
import {
  ProductCard,
  ProductCardActions,
  ProductCardContent,
  ProductCardImage,
} from '@faststore/ui'
```

## Best practices

### Do's
- Use high-quality images
- Use the ProductCard to get the primary content of a product

### Don'ts
- For empty state, if the case use the `Skeleton` component.

## Accessibility
- The `img` elements must have an `alt` attribute. 
- For images containing text, the alt description must includes the image's text.
