# SelectField

SelectField wraps a [Select](/reference/ui/atoms/Select) input and its corresponding [Label](/reference/ui/atoms/Label). It allows users to select one option from a set.

## Overview

```tsx live
<div className="overviewSection">
  <SelectField
    id="select-overview-default"
    label="Select a Status:"
    options={{
      great: 'Great',
      ok: 'Ok',
      bad: 'Bad',
    }}
  />
  <SelectField
    id="select-overview-disabled"
    label="Select a Status:"
    options={{
      great: 'Great',
      ok: 'Ok',
      bad: 'Bad',
    }}
    disabled
  />
</div>
```

---

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { SelectField } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/molecules/SelectField/styles.scss'
```

---

## Usage

```tsx live
<SelectField
  id="select-order-by"
  label="Order by:"
  options={{
    name_asc: 'Name, A-Z',
    name_desc: 'Name, Z-A',
    top_sales: 'Top Sales',
  }}
/>
```
