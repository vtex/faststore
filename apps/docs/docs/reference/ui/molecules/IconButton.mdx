# IconButton

Icon Buttons wraps a [Button](/reference/ui/atoms/Button) with and [Icon](/reference/ui/atoms/Icon). It can be icons that trigger some sort of action, such as adding an item to the cart.

## Overview

```tsx live
<div style={{ display: 'flex', columnGap: '22px' }}>
  <IconButton icon={<ShoppingCart />} aria-label="Buy" variant="primary" />
  <IconButton icon={<ShoppingCart />} variant="primary">
    Buy Now
  </IconButton>
</div>
```

## Best practices

### ✅ Do's

- Use icon-only buttons only for widespread actions with highly standardized icons, such as a cross for close.
- Use the `aria-label` attribute to provide a textual alternative for icon-only buttons.

### ❌ Don'ts

- Don't rely on icons to communicate complex actions.
- Avoid using `aria-label` when the button already has a label.
