# ButtonBuy

ButtonBuy intends to send the user to the checkout/cart step directly.

## Overview

```tsx live
<ButtonBuy>Buy Now</ButtonBuy>
```

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { ButtonBuy } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/atoms/ButtonBuy/styles.scss'
```

## Usage

```tsx live
<ButtonBuy>Buy Now</ButtonBuy>
```
