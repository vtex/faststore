
## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { Toggle } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/molecules/Toggle/styles.scss'
```
