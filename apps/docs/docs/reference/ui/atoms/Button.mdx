# Button

Buttons indicate actions that users can take, such as adding an item to the cart, submitting a form, and proceeding with an operation.

## Import

```tsx
import { Button } from '@faststore/ui'
```

## Usage

```tsx live
<Button>Hello</Button>
```

<PropsSection name="Button" />

## Customization
