# Incentive

The `Incentive` component is a badge or seal used to build trust among your potential customers. 

## Import

```tsx
import { Incentive } from '@faststore/ui'
```

## Usage

```tsx live
<Incentive>
  <SafetyIcon />
  <span>
    Trusted by SafeCompany
  </span>
</Incentive>
```

## Props

The `Incentive` component supports all attributes supported by the `<div>` tag. Besides those attributes, the following props are also supported:

<PropsSection name="Incentive" />

## Customization

`data-fs-incentive`
