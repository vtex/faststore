# List

Lists group related content. They can be ordered or unordered.

## Overview

```tsx live
<div style={{ display: 'flex', columnGap: '60px' }}>
  <List marker>
    <li>Great</li>
    <li>Ok</li>
    <li>Bad</li>
  </List>

  <List variant="ordered" marker>
    <li>Great</li>
    <li>Ok</li>
    <li>Bad</li>
  </List>

  <List>
    <li>Great</li>
    <li>Ok</li>
    <li>Bad</li>
  </List>
</div>
```

## Design Tokens

### Variants

#### Unordered

```tsx live
<List marker>
  <li>Great</li>
  <li>Ok</li>
  <li>Bad</li>
</List>
```

#### Ordered

```tsx live
<List variant="ordered" marker>
  <li>Great</li>
  <li>Ok</li>
  <li>Bad</li>
</List>
```

#### Without marker

```tsx live
<List variant="ordered">
  <li>Great</li>
  <li>Ok</li>
  <li>Bad</li>
</List>
```
