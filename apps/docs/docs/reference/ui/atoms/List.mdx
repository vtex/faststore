# List

Lists group related content. They can be ordered or unordered.

## Overview

```tsx live
<div style={{ display: 'flex', columnGap: '60px' }}>
  <List marker>
    <li>Great Unordered List</li>
    <li>Ok Unordered List</li>
    <li>Bad Unordered List</li>
  </List>

  <List as="ol" marker>
    <li>Great Ordered List</li>
    <li>Ok Ordered List</li>
    <li>Bad Ordered List</li>
  </List>

  <List>
    <li>Great</li>
    <li>Ok</li>
    <li>Bad</li>
  </List>
</div>
```

### Variants

#### Unordered

```tsx live
<List marker>
  <li>Great Unordered List</li>
  <li>Ok Unordered List</li>
  <li>Bad Unordered List</li>
</List>
```

#### Ordered

```tsx live
<List as="ol" marker>
  <li>Great Ordered List</li>
  <li>Ok Ordered List</li>
  <li>Bad Ordered List</li>
</List>
```

#### Without marker

```tsx live
<List as="ol">
  <li>Great</li>
  <li>Ok</li>
  <li>Bad</li>
</List>
```
