# Radio

Radios allow users to select one option from a set.

## Overview

```tsx live
<div style={{ display: 'flex', columnGap: '16px' }}>
  <Radio />
  <Radio checked readOnly />
  <Radio disabled />
  <Radio checked disabled readOnly />
</div>
```

<small>
  You can remove <code>readOnly</code> prop when using.
</small>

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { Radio } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/atoms/Radio/styles.scss'
```

## Usage

```tsx live
<Radio />
```
