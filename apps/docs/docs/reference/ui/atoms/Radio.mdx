# Radio

Radios allow users to select one option from a set.

## Import

```tsx
import { Radio } from '@faststore/ui'
```

### Import Styles

```tsx
import '@faststore/ui/src/components/atoms/Radio/styles.scss'
```

## Usage

```tsx live
<Radio />
```

## Props

<PropsSection name="Radio" />

## Customization

For futher customization, use the data-attr below:

`data-fs-radio`
