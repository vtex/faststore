# Input

Inputs are text fields used to get user inputs.

## Import

```tsx
import { Input } from '@faststore/ui'
```

## Usage

```tsx live
<Input
  placeholder="Input"
/>
```

## Variants

### Success

```tsx live
<Input variant="success" placeholder="Success"/>
```

### Error

```tsx live
<Input variant="error" placeholder="Error"/>
```
