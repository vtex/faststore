Link performs the same behavior of a regular anchor tag (`a`), but can be used as any HTML tag and component, including specific Frameworks link components.

## Overview

```tsx live
<div className="overviewSection">
  <Link variant="default">Default</Link>
  <Link variant="display">Display</Link>
  <Link variant="inline">Inline</Link>
  <Link size="small">Small</Link>
<div style={{ backgroundColor: '#171a1c', padding: '0.5rem' }}>
   <Link inverse={true}>Inverse Link</Link>
 </div>
</div>
```

---

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { Link } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/atoms/Link/styles.scss'
```

---

## Usage

```tsx live
<Link href="/">
    A default link
</Link>
```

### Props

The `Link` component supports all attributes supported by the tag chosen through the `as` prop. 
For example, if someone wants to use a wrapper for any external link component (i.e., Next.js `Link` or Gatsby `Link`), the FastStore UI `Link` component can be used with the `as="div"` prop.

Besides those attributes, the following props are also supported:

<PropsComponent
  name={'testId'}
  type={'string'}
  defaultValue={'fs-link'}
  description={'ID to find this component in testing tools (e.g.: cypress, testing library, and jest).'}
  required={false}
/>
<PropsComponent
  name={'variant'}
  type={'"default" | "display" | "inline"'}
  defaultValue={'default'}
  description={'Specifies the component variant.'}
  required={false}
/>
<PropsComponent
  name={'inverse'}
  type={'boolean'}
  defaultValue={false}
  description={'Defines use of inverted color.'}
  required={false}
/>
<PropsComponent
  name={'size'}
  type={'"small" | "regular"'}
  defaultValue={'regular'}
  description={'Defines size of element'}
  required={false}
/>

## Customization

For further customization, you can use the following data attributes:

`data-fs-link`

`data-fs-link-variant="default"`

`data-fs-link-variant="display"`

`data-fs-link-variant="inline"`

`data-fs-link-size="small"`

`data-fs-link-inverse="true"`
