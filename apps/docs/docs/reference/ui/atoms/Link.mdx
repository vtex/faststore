Link performs the same behavior of a regular anchor tag (`a`), but can be used as any HTML tag and component, including spacifics Frameworks link components.

## Overview

```tsx live
<div style={{ display: 'flex', columnGap: '16px', alignItems: 'center' }}>
  <Link variant="default">Default</Link>
  <Link variant="display">Display</Link>
  <Link variant="footer">Footer</Link>
  <Link variant="inline">Inline</Link>
  <Link inverse={true}>Inverse</Link>
</div>
```

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { Link } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/atoms/Link/styles.scss'
```

## Usage

```tsx live
<Link href="/">
    A default link
</Link>
```

### As `div`

```tsx live
<Link as="div">
    <span>A span inside the Link component</span>
    <a href="/">A link inside the Link component</a>
</Link>
```

## Props

The `Link` component supports all attributes supported by the tag chosen through the `as` prop. 
For example, if someone wants to use a wrapper for any external link component (i.e., Next.js `Link` or Gatsby `Link`), the FastStore UI `Link` component can be used with the `as="div"` prop.
