# Slider

Sliders allow users to select a single value from a range.

## Import

```tsx
import { Slider } from '@faststore/ui'
```

## Usage

```tsx live
<Slider
  max={500}
  min={0}
/>
```

## Props

<PropsSection name="Slider" />

## Customization

`data-fs-slider`

`data-slider-range`

`data-slider-thumb='(left|right)'`
