# Checkbox

Checkboxes allow users to toggle an option on or off.

## Overview

```tsx live
<div style={{ display: 'flex', columnGap: '16px' }}>
  <Checkbox />
  <Checkbox checked readOnly />
  <Checkbox disabled />
  <Checkbox checked partial readOnly />
</div>
```

<small>
  You can remove <code>readOnly</code> prop when using.
</small>

---

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { Checkbox } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/atoms/Checkbox/styles.scss'
```
