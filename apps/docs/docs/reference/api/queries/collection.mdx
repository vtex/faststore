# collection

Returns the details of a product based on the specified locator.

## Usage

<div className="lg:grid grid-cols-2 gap-20">
  <div>
    <h3 className="mb-4">Arguments</h3>

  <PropsComponent
      name="slug"
      type="String"
      description="Slug of the collection you want to query."
      required="true"
    />

</div>
<div>

<Tabs>
  <TabItem value="query" label="Query" default>

```graphql
query ($slug: String!) {
  collection (slug: $slug) {
    # StoreCollection fields
  }
}
```

</TabItem>

<TabItem value="response" label="Response">

The `collection` query returns a `StoreCollection`. To learn more about each field that can be queried check the <a href="/reference/api/objects#storecollection">`StoreCollection`</a> reference.

</TabItem>
</Tabs>
</div>
</div>


## Example

Get the ID, type, SEO title and selected facets - with key and value - of the collection with slug `office`.

<GraphQLExplorer query='query {
  collection (slug:"office") {
    id,
    type,
    seo {
      title
    }
    meta {
      selectedFacets {
        key,
        value
      }
    }
  }
}'/>
