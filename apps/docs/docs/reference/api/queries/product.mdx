# `product`

Returns the details of a product based on the specified locator.

<div className="grid grid-cols-2 gap-20">

<div>

## Usage

### Arguments

<PropsComponent
      name="locator"
      type="IStoreSelectedFacet"
      description="An array of selected search facets, each being an object with two properties `key` and `value`."
      required="true"
    />

<PropsComponent
      name="IStoreSelectedFacet.key"
      type="String"
      description="<a href="/reference/api/inputs#istoreselectedfacet">`ISelectedStoreFacet`</a> property. To query information of a product by its SKU ID, use `key:"id"`."
      required="true"
    />

<PropsComponent
      name="IStoreSelectedFacet.value"
      type="String"
      description="<a href="/reference/api/inputs#istoreselectedfacet">`ISelectedStoreFacet`</a> property. To query information of a product that has SKU ID `123`, use `value:"123"`."
      required="true"
    />

### Fields

The `product` query returns a `StoreProduct`. To learn more about each field that can be queried check the <a href="/reference/api/objects#storeproduct">`StoreProduct`</a> reference.

</div>
<div>

**Query**

```graphql
query Product($locator: [IStoreSelectedFacet!]!) {
  product(locator: $locator) {
    # StoreProduct fields
  }
}
```

**Example**

Getting the name, description, brand name and image URL and alternate name of a product with SKU of `123`.

```graphql
query {
  product (locator:[{key:"id", value:"123"}]) {
    name,
    description,
    brand {
      name
    },
    image{
      url
    }
  }
}
```
