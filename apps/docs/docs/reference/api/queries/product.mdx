# product

Returns the details of a product based on the specified locator.

## Usage

<div className="lg:grid grid-cols-2 gap-20">

<div>

      description="An array of selected search facets."
      required="true"
/>

<PropsComponent
      name="key"
      type="String"
      description="<code>IStoreSelectedFacet</code> property. To query information of a product by its SKU ID, use <code>key:'id'</code>."

/>

<PropsComponent
      name="value"

## Example

Get the name, description, brand, image URL, and image alternate name of the product with the `97730863` SKU.

<GraphQLExplorer query='query {
  product (locator:[{key:"id", value:"97730863"}]) {
    name,
    description,
    brand {
      name
    },
    image{
      url,
      alternateName
    }
  }
}'/>
