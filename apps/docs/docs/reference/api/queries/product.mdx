## `allProducts`

Returns information about all products.

### Usage

#### Arguments

<PropsComponent
      name="first"
      type="Int"
      description="	Number of items that should return from the complete result list."
      required="true"
    />

<PropsComponent
      name="after"
      type="String"
      description="Product pagination argument, indicating the cursor corresponding with the item after which the results should be fetched."
    />

#### Fields

Below you can see a list of all fields that can be queried with `allProducts`. Click on the links to learn more details about each field.

- <a>StoreProductConnection</a>
  - pageInfo: <a href="/reference/api/objects/#StorePageInfo">StorePageInfo</a>
    - hasNextPage: boolean!
    - hasPreviousPage: boolean!
    - startCursor: string!
    - endCursor: string!
    - totalCount: int!
  - edges: [<a href="/reference/api/objects/#StoreProductEdge">StoreProductEdge</a>]!
    - node: <a href="/reference/api/objects/#StoreProduct">StoreProduct</a>!
      - Product information. Check <a href="/reference/api/objects/#StoreProduct">StoreProduct</a> to see all possible fields.
    - cursor: string!


</div>
<div>


**Query**

```graphql
query($first: Int!){
    allProducts(first: $first) {
      # StoreProductConnection fields
    }
}
```

**Example**

Getting the names, SKUs and cursors of the first five products after the product associated with cursor `"4"`. Also getting the last cursor on the list and whether there are more results for next page.

```graphql
query {
  allProducts (first:5, after:"4") {
    pageInfo {
      hasNextPage,
      endCursor
    },
    edges {
      node {
        name,
        sku
      },
      cursor
    }
  }
}
```
