**Query**

```graphql
query($first: Int!){
    allProducts(first: $first) {
      # StoreProductConnection fields
    }
}
```

**Example**

Getting the names, SKUs and cursors of the first five products after the product associated with cursor `"4"`. Also getting the last cursor in the complete list and whether there are more results for next page.

```graphql
query {
  allProducts (first:5, after:"4") {
    pageInfo {
      hasNextPage,
      endCursor
    },
    edges {
      node {
        name,
        sku
      },
      cursor
    }
  }
}
```
