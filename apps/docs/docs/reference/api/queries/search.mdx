**Query**

```graphql
query ($first: Int!){
  search (first: $first) {
    product {
      # StoreProductConnection fields
    },
    suggestions {
      # StoreSuggestions fields
    },
    facets {
      # StoreFacet fields
    }
  }
}
```

**Example**

- Getting the keys, labels, values' labels and values' quantities from the first three facets in the returned list, searching with the term `"wooden"`.

```graphql
query {
  search (first:3, term:"wooden") {
    facets {
      key,
      label,
      values {
        quantity
      }
    }
  }
}
```

- Getting cursors, names, SKU IDs, and image URLs for the first 10 products after the product with cursor `"10"`, searching by the term `"steel"`. Also getting the total quantity of results in the complete list and whether there are more results for next page.

```graphql
query {
  search (first: 10, after: "10", term:"steel") {
    products {
      pageInfo {
        hasNextPage,
        totalCount
      },
      edges {
        node {
          name,
          sku,
          image {
            url
          }
        },
        cursor
      }
    }
  }
}
```

- Getting terms' values, terms' counts, products' names and products' SKU IDs for the first five sugestions from a list sorted from higher to lower discount, searching by the term `"table awesome"`.

```graphql
query {
  search (first:5, sort: discount_desc, term:"table awesome") {
    suggestions {
      terms {
        value,
        count
      },
      products {
        name,
        sku
      }
    }
  }
}
```

- Getting information about facets, products and suggestions at once.

```graphql
query {
  search (first:5, after:"5", term:"wooden", sort:score_desc) {
    products {
      pageInfo {
        totalCount,
        hasNextPage,
        endCursor
      },
      edges {
        node {
          name,
          sku,
          slug,
          image {
            url,
            alternateName
          }
        },
        cursor
      }
    },
    suggestions {
      terms {
        value,
        count
      },
      products {
        name,
        slug,
        description
      }
    },
    facets {
      key,
      label,
      values {
        value,
        selected
      },
      type
    }
  }
}
```
