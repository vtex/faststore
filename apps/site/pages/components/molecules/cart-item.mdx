## Overview

export function priceFormatter(price) {
  return useMemo(() => {
    const formattedPrice = new Intl.NumberFormat('en-US', {
      style: 'currency',
      currency: 'USD',
    }).format(price)
    return formattedPrice
  })
}

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { CartItem } from '@faststore/ui'
```

Import Styles


## Customization

For further customization, you can use the following data attributes:

`data-fs-cart-item`

`data-fs-cart-item-content`

`data-fs-cart-item-image`

`data-fs-cart-item-title`

`data-fs-cart-item-skus`

`data-fs-cart-item-remove`

`data-fs-cart-item-prices`

`data-fs-cart-item-actions`
