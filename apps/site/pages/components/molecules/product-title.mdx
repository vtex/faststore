# Product Title

The Product Title is commonly used on Product Details Page (PDP). It usually wraps the product name and a product label.

</header>

## Overview

## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { ProductTitle } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/molecules/ProductTitle/styles.scss'
```
