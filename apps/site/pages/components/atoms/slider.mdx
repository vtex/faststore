## Import

Import the component from <a href="https://www.faststore.dev/reference/ui/get-started-faststore-ui">@faststore/ui</a>

```tsx
import { Slider } from '@faststore/ui'
```

Import Styles

```tsx
import '@faststore/ui/src/components/atoms/Slider/styles.scss'
```

---

export function priceFormatter(price) {
  return useMemo(() => {
    const formattedPrice = new Intl.NumberFormat('en-US', {
      style: 'currency',
      currency: 'USD',
    }).format(price)
    return formattedPrice
  })
}

## Usage

<OverviewSection>
  <Slider
    max={{ absolute: 500, selected: 250 }}
    min={{ absolute: 0, selected: 100 }}
    maxValueLabelComponent={(maxValue) => {
      return <span>{maxValue}</span>
    }}
    minValueLabelComponent={(minValue) => {
      return <Price formatter={priceFormatter} value={minValue} />
    }}
  />
</OverviewSection>
```tsx
<Slider

  min={{ absolute: 0, selected: 100 }}
```

## Customization

For further customization, you can use the following data attributes:

`data-fs-slider`

`data-fs-slider-range`

`data-fs-slider-thumb="left" | "right"`
