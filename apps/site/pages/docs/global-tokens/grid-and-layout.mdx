## Breakpoints

The following tokens are used as a mirror for the breakpoint system, and they can be used to create a component that has a width related to it  (see [Modal](/components/molecules/modal#design-tokens) `--fs-modal-max-width` as an example).

<Callout type="warning" emoji="⚠️">
  We do not recommend to change theses tokens, if you want to override these sizing system, check the [Mixins & Breakpoints](/docs/customizing/mixins-breakpoints) page.
</Callout>


```scss
$breakpoints: (
  "phone":         320px,
  "phonemid":      375px,
  "tablet":        768px,
  "notebook":      1280px,
  "desktop":       1440px,
);
```
