# Dashboard

</header>

<Callout type="warning" emoji="⚠️">
  This documentation is currently under development.
</Callout>

After completing the deployment process and successfully creating both the repository and the live preview of the store, you can access the Dashboard page to view more information about your store deploys.

On the Dashboard, you can check the current status of your store, including its domains, deployed versions, store preview, and settings.

The Dashboard page consists of two main views: Overview and Deploys. Refer to the details provided above for more information on each view.
