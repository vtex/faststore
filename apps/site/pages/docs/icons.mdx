<header>

# Icons

Icons help build web pages by illustrating concepts and improving website navigation.

</header>

FastStore offers a set of icons that our components use, and you can also use them on your application. This set of icons uses SVGs from [Phosphor icons](https://phosphoricons.com/).

---

## Import

Import the component from [@faststore/ui](/../components)

```tsx copy
import { Icon } from '@faststore/ui'
```

You can find more details about the `Icon` component [here](/components/atoms/icon).

## Usage

<OverviewSection>
  <Icon name="X" weight="bold" />
  <Icon name="MagnifyingGlass" weight="regular" />
  <Icon name="House" weight="thin" />
</OverviewSection>

```tsx copy
<Icon name="X" weight="bold" />
<Icon name="MagnifyingGlass" weight="regular" />
<Icon name="House" weight="thin" />
```
