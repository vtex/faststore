## Components

The `Dropdown` uses the [Compound Component](https://kentcdodds.com/blog/compound-components-with-react-hooks) pattern, its components are:

- `Dropdown`: the wrapper component, it can receive a `isOpen` and emit the event `onDismiss`, can be used as controleed component.
- `DropdownButton`: the component that emit the trigger to open de DropdownMenu.
- `DropdownItem`: Option for DropdownMenu.
- `DropdownMenu`: the wrapper to dropdownItems.
