<header>

# Global Design Tokens

</header>

Global design tokens are parameters that affect all UI look 'n' feel. They are the main configuration file of a Theme.
Component-specific tokens often inherit from global tokens but are named specifically to apply those tokens in component development.

- [Spacing](#spacing)
- [Grid & Layout (Padding, Container, Gaps, Breakpoints)](#grid-layout)
- [Interactive Controls (Tap, States, Sizing)](#interactive-controls)
- [Refinements (Transition, Borders, Shadow)](#refinements)
