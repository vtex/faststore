## Description

This plugin integrates gatsby-theme-store with Facebook Pixel.

## How to install

Just run `yarn add @vtex/gatsby-plugin-pixel-facebook` at the root folder of your store.

## Available options

- pixelId **required**: the ID of the Facebook pixel.
